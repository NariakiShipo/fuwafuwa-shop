# 📚 FUWA FUWA Pet Shop - 文件索引

## 🎯 關於此專案

FUWA FUWA Pet Shop 是一個具有可愛設計和互動功能的寵物商店 Web 應用程式。
使用 React + TypeScript 建置，採用基於參考圖片的溫暖配色和圓角設計。
