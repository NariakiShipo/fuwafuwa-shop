# 📚 FUWA FUWA Pet Shop - 文件索引

## 🎯 關於本專案

FUWA FUWA Pet Shop 是一個具有可愛設計和互動功能的寵物商店網頁應用程式。
使用 React + TypeScript 建置，採用參考圖片為基礎的溫暖配色和圓角設計。
